## Overview

The call transfer connector helps Natalia transfer calls to human agents when needed. It's designed to be simple to set up while providing flexible routing options.

## Configuration in Dashboard

### Basic Setup
1. **Phone Numbers**
  - Follow format: +XXXNUMBER (e.g., +33612345678)
  - Add transfer destinations
  - Define names and roles

2. **Transfer Rules**
  - Set routing conditions
  - Define priority levels
  - Configure availability schedules (Coming soon)

### Transfer Types
- Direct to specific agent
- Team/department routing
- Skill-based routing
- Fallback options

## Best Practices

1. **Clear Communication**
   * Inform customer about transfer
   * Explain transfer reason
   * Set wait time expectations
2. **Smooth Handoffs**
   * Check agent availability
   * Preserve conversation context
   * Confirm successful transfer
3. **Quality Assurance**
   * Monitor transfer success rates
   * Track customer satisfaction
   * Review transfer patterns
